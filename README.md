# Job Recruitment Management System

A multi-role job recruitment website developed as a university project.

## Technologies

- ASP.NET MVC
- Entity Framework
- LINQ
- SQL Server
- HTML
- CSS
- JavaScript
- Bootstrap

## Features

### Candidate
- Register / Login
- Search jobs
- Filter jobs
- Save jobs
- Submit CV
- Manage profile

### Employer
- Company profile
- Create job posts
- View candidate CVs
- Candidate recommendation

### Admin
- User management
- Job management
- CV template management
- Recruitment guide management
- Dashboard and statistics
